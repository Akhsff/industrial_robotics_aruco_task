# Запуск проекта (1 вариант)

## Подключение к камере через ROS2
###### в первом терминале: 
ros2 launch pylon_ros2_camera_wrapper pylon_ros2_camera.launch.py
###### во втором терминале: 
ros2 run rqt_image_view rqt_image_view
###### сохраняем изображение с камеры под названием *image.png*
###### во третьем терминале (при необходимости можно поменять название изображения в коде):
python3 process_aruco_pose.py
###### на выходе получаем текстовый с названием *marker_pose_goal.txt*, который содержит координаты *x* и *y*
###### в четвертом терминале (скрипт автомотический считывает координаты из текстового файла): 
python3 kuka_point.py
